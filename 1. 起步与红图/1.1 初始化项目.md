### 2.初始化Flask核心对象
app/app.py

```python
from flask import Flask

__author__ = "gaowenfeng"


def create_app():
    app = Flask(__name__)
    app.config.from_object('app.config.secure')
    app.config.from_object('app.config.setting')

    return app
```

ginger.py
```python
from app.app import create_app

__author__ = "gaowenfeng"

app = create_app()

if __name__ == '__main__':
    app.run(debug=True)
```
